# Default Service Ports 

For both Single node and multi-node deployments following ports are used. All services exposing APIs will use the below ports

```yaml
CMS: 30445
AAS: 30444
SCS: 30501
SHVS: 30500
SQVS: 30502
IHUB: None
KBS: 30448
K8s-scheduler: 30888
K8s-controller: None
SKC Library: None
SGXAgent: 31001
```
